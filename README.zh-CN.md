清理一些 twitter 上烦人的元素，让你的 twitter 体验更加干净。

🌟 Features

- 隐藏首页的标签页
- 隐藏主页右侧边栏
- 隐藏搜索和发现
- 隐藏获得认证的广告
- 隐藏发现更多
- 隐藏搜索页面的推荐
- 隐藏查看推文分析链接
- 隐藏其他的广告
- 隐藏直播
- 恢复 Twitter 的 Logo

## 局限性

由于 Firefox 默认不支持 CSS 特性 `:has`，所以需要手动启用它，参考：<https://developer.mozilla.org/en-US/docs/Web/CSS/:has#browser_compatibility>
